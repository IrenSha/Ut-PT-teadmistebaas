# Paide Gümnaasiumi uurimistöö ja praktilise töö teadmistebaas

See teadmistebaas põhineb Paide Gümnaasiumi uurimistöö ja praktilise töö ametlikul juhendil ning aitab õpilastel mõista ja leida juhendi nõudeid uurimistöö, praktilise töö, vormistamise, viitamise, APA7, hindamiskriteeriumide ja töö ülesehituse kohta.

## Teemad
*Teemad lisanduvad järk-järgult.*

- Uurimistöö ja praktilise töö ülesehitus
- [Uurimistöö ja praktilise töö vormistamine](ptk-05-vormistamine.md)
- Allikaloend ja APA7 juhised
- Hindamiskriteeriumid

See teadmistebaas on koostatud Paide Gümnaasiumi õpilaste toetamiseks uurimistöö ja praktilise töö koostamisel ning põhineb Paide Gümnaasiumi ametlikul uurimistöö ja praktilise töö juhendil.
